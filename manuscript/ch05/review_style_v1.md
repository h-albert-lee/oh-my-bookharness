- ::: note '파라메트릭 메모리와 비파라메트릭 메모리' 블록 내부에 'ch05'라는 내부 작업 코드가 그대로 노출되어 있음('이 구분이 ch05 전체 논의의 뼈대가 된다'). 출판 원고에 노출될 수 없는 초안 메모이므로 '이 구분이 이 장 전체 논의의 뼈대가 된다'로 교체해야 함.
- 5.2.2절 본문이 '문서의 논리 단위를 무시하고 고정 크기로 자르면,' 문장 중간에서 미완성 상태로 끊겨 있음. 초안 미완성 구간이므로 원고 제출 전 반드시 완성하거나 '[작성 중]' 표시를 명시해야 함.
- 5.2.1절에서 '3장에서 다룬 논지'를 참조하고 있으나('이 상황은 3장에서 다룬 논지와 같은 구조를 가진다'), 해당 내용이 실제로 3장에 존재하는지 확인되지 않음. 전후 장 참조는 원고 전체 구성이 확정된 후 검증하고, 미확정 상태라면 각주 또는 편집자 메모로 처리해야 함.
- 5.2절 도입부의 그림 5-1은 실제 다이어그램 없이 HTML 주석(<!-- figure: ... -->)으로만 대체되어 있음. 텍스트 기반 ASCII 다이어그램 또는 실제 그림 파일로 교체해야 하며, 주석 형태로 원고를 제출할 수 없음.
