# 출판사 원고 형식 가이드 (한빛미디어)

## 위계 구조

출판 원고의 위계는 다음과 같다. 장 번호가 N일 때:

```
N장. (장제목)           → # N장. 장제목
  N.1. (절제목)         → ## N.1 절제목
    N.1.1 (중제목)      → ### N.1.1 중제목
      소제목            → #### 소제목
        소소제목        → ##### 소소제목
```

### 규칙
- 모든 ##, ### 레벨 헤딩에는 반드시 넘버링을 포함한다.
- 넘버링 없는 ##, ### 헤딩은 허용하지 않는다.
- 각 절(##)에는 최소 2개의 중제목(###)이 포함되어야 한다.
- #### 이하는 넘버링 없이 사용할 수 있다.

## 구조물 형식

본문 중 특수 구조물은 다음 형식을 사용한다:

### 박스 (참고사항, 심화 설명, 사례 분석)
본문 흐름에서 분리되는 내용. 편집 시 별도 박스로 디자인된다.
```markdown
::: box 제목
내용
:::
```

### 노트 (용어 정의, 개념 보충)
짧은 보충 설명. 편집 시 노트/팁 스타일로 디자인된다.
```markdown
::: note 제목
내용
:::
```

### 팁 (실무 팁, 권장사항)
실무적 조언. 편집 시 팁 스타일로 디자인된다.
```markdown
::: tip 제목
내용
:::
```

### 금지 형식
- `> **[참고]**` 같은 blockquote 기반 구조물 사용 금지
- `[참고사항]`, `[용어 정의]` 같은 비표준 인라인 마커 금지
- blockquote(`>`)는 인용문에만 사용

## 표와 그림

### 표
- 캡션은 표 위에 배치: `**표 N-M. 제목**`
- Markdown 표 형식 사용

### 그림
- 캡션은 그림 아래 배치: `**그림 N-M. 제목**`
- ASCII 다이어그램 또는 `<!-- figure: 그림 N-M. 제목 -->` 플레이스홀더 사용
- 편집 단계에서 정식 도판으로 교체
